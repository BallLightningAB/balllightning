**Project: Web-Development - Jorild.se**

May 2025 - Jun 2025

Associated with Ball Lightning AB

https://github.com/BallLightningAB/jorild-se

Designed and developed a modern, high-performance bilingual website for Jorild Osteopati, an osteopathy clinic in Sweden.



Technology: Next.js 15, React 19, TypeScript, Tailwind CSS, ShadCN UI, Edge Runtime.



Features:



WCAG 2.1 AA accessibility



Swedish/English with geolocation-based detection and manual switch



Core Web Vitals–optimized with 100/100 Lighthouse SEO score



Structured data for improved search visibility



Performance: Edge-optimized middleware, automatic image and font optimization, code splitting, and lazy loading.



The result is a conversion-focused, SEO-friendly platform that delivers a seamless experience on both desktop and mobile, coupled with a new Google Business Profile.



Skills: Web Design · Online Marketing · Agile Project Management · Technical Project Management · Artificial Intelligence (AI)
